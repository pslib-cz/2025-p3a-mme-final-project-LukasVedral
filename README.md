# Projekt: Solo Grip (Jednoruční Hangboard)

Kompaktní, přenosná tréninková pomůcka pro izolované posilování stisku jedné ruky. Konstrukce využívá 3D tištěné tělo s variabilní hloubkou lišt a integrovaným závěsným okem pro upevnění na lano. Cílem je maximální skladnost při zachování ergonomie pro různé typy úchopů.
